Falcon-Pad Documentation

Interface guide and feature overview. Everything you need to get started.

Step 1

Launcher Window

When you run FalconPad.exe, a small tray window opens showing your connection info.

  • LOCAL — open in a browser on the same PC
  • NETWORK — open on your tablet or phone (same network)
  • Falcon BMS — detection status, goes green when BMS is running
  • ACMI — aircraft contacts status (set g_bTacviewRealTime 1 required)
  • LOGS — path to the log file for troubleshooting
  • PUSH TO EJECT — stops the server
Map

Tactical Map

The main view. Your aircraft, steerpoints, threat rings, bullseye and all contacts are overlaid on a live map.

  • PPT CIRC — toggle threat range circles
  • PPT NAME — toggle threat labels
  • APRI / ICAO / RWY — airport info overlays
  • Bullseye ring — toggle bullseye display
  • N/S — North-up / track-up toggle
  • Layers — switch between dark, satellite and terrain tiles

Top toolbar: draw tools, measure, link share, clear, and color picker.

GPS

GPS & Steerpoints

The GPS tab shows your live flight data and steerpoint list.

  • LAT / LON — current position in decimal degrees
  • HDG — magnetic heading
  • ALT — altitude in feet MSL
  • KIAS — indicated airspeed in knots
  • DIST / STPT — distance to next steerpoint
  • BULL — bearing and range from bullseye

Steerpoints are listed below with index, name and altitude. Auto-loaded from BMS or your DTC .ini file.

Briefing

Briefing Viewer

Import and read mission briefings directly on your tablet.

  • Supports PDF, images, DOCX and HTML
  • Pinch-to-zoom on tablets
  • Click IMPORT to upload a file from your device
  • Or drop files directly into the personal/ folder next to the exe — they appear automatically
  • BMS-generated briefings are loaded automatically when available
Charts

Falcon Charts

The Charts tab embeds Falcon-Charts directly in the interface — airport charts, approach plates and procedures for the current theater.

  • Search by airport name or ICAO code
  • Dark charts optimized for low-light cockpit use
  • Theater auto-selected based on your current BMS session
Kneeboard

Kneeboard & Checklists

The Kneeboard tab centralizes all mission reference material in one place.

  • CHECKLIST — F-16 checklist from T.O. BMS 4.38, 215 items across all flight phases. Tap to check off items, RESET to start over
  • FLIGHT PLAN — steerpoint list with altitudes and legs
  • NOTES — free-text notes pad
  • 9-LINE — CAS 9-line form
  • BREVITY — quick reference brevity codes

Checklist phases are color-coded: ground, taxi, takeoff, combat, landing, shutdown.

Settings

Settings Panel

Click the gear icon in the status bar to open settings.

  • Listen Port — default 8000, restart required after change
  • Broadcast Interval — position update rate in ms (100 = smooth, 500 = eco)
  • Map colors — customize ownship, ally, enemy and airport colors
  • Steerpoints — line and point size for steerpoint display
  • Flight Plan — line and point size for flight plan legs
  • PPT Threats — circle thickness and point size for threat rings
  • Bullseye — color and ring thickness toggle
